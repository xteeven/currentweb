# Bio

Computer Scientist, Mechatronic Engineer, and Ph.D. Researcher in the Human-Centered Ubiquitous Media group at LMU Munich. My research field is human-computer interaction, particularly human augmentation, human-computer integration, and human-computer symbiosis. I'm highly interested in creating and understanding technologies that help humans overcome their natural sensory, cognitive, and motor limitations.
